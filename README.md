## 📦 How to Run Locally
1. Clone this repo:
   ```bash
   git clone https://github.com/Wasif-Karim03/Portfolio.git
   cd Portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
